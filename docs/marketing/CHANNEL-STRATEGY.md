## Content/SEO Strategy (Primary Channel)

### Why Content/SEO
- Developer/technical audience researches before buying
- Shopify app development queries have strong search intent
- Long-term sustainable pipeline vs. paid churn
- Supports both ICPs (merchants + developers)

### Content Pillars

1. **Educational (Top of Funnel)**
   - "How to build a Shopify app" (comprehensive guide)
   - "Shopify API tutorial" series
   - Build logs and technical deep-dives

2. **Comparison/Intent (Mid Funnel)**
   - "Best Shopify app development agencies" (we rank vs. competitors)
   - "Shopify app studio vs. agency" decision guide

3. **Social Proof (Bottom of Funnel)**
   - Case studies (post-launch)
   - Customer testimonials
   - Build retrospective posts

4. **Developer-Focused**
   - Open source project documentation
   - API reference guides
   - Integration tutorials

### Content Cadence
- 2 long-form articles/month (one merchant-focused, one developer-focused)
- Build log updates as apps ship
- Technical tutorials as we solve interesting problems

### Expected Timeline
- Month 1-3: Foundation (site SEO, first 4-6 articles)
- Month 4-6: Early ranking, 500-1000 organic sessions/month
- Month 6-12: Meaningful organic pipeline, first organic MQLs

## Developer Community Strategy

### Target Communities
- r/shopify (Reddit)
- Shopify Community forums
- Hacker News
- Indie Hackers
- Relevant Discord servers

### Approach
- Participate genuinely, not just self-promote
- Share build logs and lessons learned
- Open source tooling that developers find useful
- Answer technical questions where we have expertise

### Success Metrics
- Community mentions/reputation
- Referral traffic from community links
- Direct inquiries from community members

---

## Partnerships Strategy

### Shopify Partner Program
- Enroll as Shopify Partner
- Build apps for App Store consideration
- Co-marketing with complementary apps (reviews, loyalty, CMS)

### Agency Partnerships
- White-label or referral arrangements with agencies who overflow work
- Focus on agencies who need specialized Shopify app expertise

## Measurement Framework

**Primary metrics:**
- Organic sessions growth
- Keyword rankings for target terms
- MQLs from organic (demo requests, discovery calls)
- Community engagement metrics

**Secondary metrics:**
- Email list growth (newsletter signups)
- GitHub stars/activity
- Partnership conversations initiated

**Reporting cadence:**
- Monthly: Content output, traffic, rankings
- Quarterly: Full channel strategy review
